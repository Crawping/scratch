Key Chain Pro

Key Chain Pro is a password management system that allows smoother integration into apps that you want to keep your passwords for. It is like KeePass, but, hopefully, better.

When you run it for the first time, it will register itself to be auto-runnable, so every time you login, it will run automatically only asking for your master password in the beginning of the session. This password is not stored anywhere ( except memory ), it is hashed using SHA-256 and a corresponding key is derived from it using AES-256 algorithm; this key is then used to encrypt and decrypt your passwords. Obviously, if you forget your master password, your other passwords will be lost ( handling this scenario is still subject to research ).

Once you have launched the app ( or it has automatically ran ), you have the following hotkeys to use:

  • Ctrl-Alt-Shift V - pastes the credentials into the currently active window. The keyboard focus should be set on the username field and the entire flow presumes a Tab keystroke will switch focus to password field.
  • Ctrl-Alt-Shift R - records credentials for a new app, alternatively, you can specify another keyname for the same app, so that whenever you paste credentials, KeyChainPro will prompt you which set of credentials you want to use.
  • Ctrl-Alt-Shift P - only paste the password ( some applications/websites remember your username and promptly specify it when you run such an app ).
  • Ctrl-Alt-Shift L - brings up the list of the currently available credentials.
  • Ctrl-Alt-Shift C - brings up the list of the currently available credentials allowing you  to choose which password to paste to clipboard.
